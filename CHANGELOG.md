# Changelog

All notable changes to memdog will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-04-26

### Added

- Multi-channel data ingestion via 33 channel adapters and 300+ app integrations (Nango)
- 42-agent AI enrichment pipeline with 6-layer deterministic classification
- Dual-layer temporal knowledge graph (Postgres + Graphiti/Neo4j)
- 5 search modes: vector, full-text, hybrid, graph, and full
- 4 reranking strategies: none, RRF, MMR, cross-encoder
- RAG chat with inline `[1][2]` citation markers
- 10 memory types with TTL, versioning, and per-item access control
- Model Garden: 16+ AI provider management with encrypted API key storage
- 5-tier smart routing (small/medium/large/multimodal/omni) with fallback chains
- DigiMe conversational AI agent for 25+ messaging platforms
- Web UI with AI Studio, playground, telemetry, and settings
- SDKs for Python, TypeScript, Go, Rust, and Ruby
- MCP server with 8 tools for Claude Desktop and Cursor
- Agent framework adapters for LangChain, CrewAI, and OpenAI
- K8s pod management from the browser (create, scale, monitor Ollama deployments)
- Multi-tenant organization hierarchy with role-based access
- Docker Compose for local development (11 services)
- Deployment guides for GKE, AWS, Azure, and Mac Mini
