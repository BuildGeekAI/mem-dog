# Welcome to Mem-Dog

Mem-Dog is a complete, self-hosted AI system that ingests data from 25+ channels and 300+ apps, enriches it with 40 specialized AI agents, stores it in a temporal knowledge graph, and makes it searchable through 5 search modes with RAG chat — all running on hardware you control.

  **New to Mem-Dog?** Start with the [Quickstart Guide](/quickstart) to get up and running in 30 seconds.
</Info>
