# Rust SDK

Fully async with tokio + reqwest. Zero-cost abstractions.

## Installation

```toml
[dependencies]
memdog-client = { git = "https://github.com/BuildGeekAI/memdog", path = "clients/rust" }
tokio = { version = "1", features = ["full"] }
```

## Usage

```rust
use mem_dog_client::MemDogClient;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = MemDogClient::new("http://localhost:8080", "md_your_key");

    // Store data
    let item = client.create_data("Meeting notes", &["standup"]).await?;

    // Search
    let results = client.semantic_search("what happened?", 10).await?;

    Ok(())
}
```
