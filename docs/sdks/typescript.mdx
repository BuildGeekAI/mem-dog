# TypeScript SDK

Type-safe client using native `fetch`. Zero external dependencies.

## Installation

```bash
npm install memdog-client
# or
yarn add memdog-client
```

## Usage

```typescript
import { MemDogClient } from 'memdog-client';

const client = new MemDogClient({
  baseUrl: 'http://localhost:8080',
  apiKey: 'md_your_key',
});

// Store data
await client.createData({
  content: 'Meeting notes from standup',
  metadata: { tags: ['standup'] },
});

// Search
const results = await client.semanticSearch({
  query: 'what happened in standup?',
  maxResults: 10,
});

// List data
const items = await client.listData({ limit: 20, tags: ['standup'] });
```

Full API coverage with TypeScript types for all request/response payloads.
